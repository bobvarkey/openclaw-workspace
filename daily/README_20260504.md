# Bob's Workspace

Personal AI operating environment, powered by OpenClaw.

## What This Is

A sprawling automation hub for a neurologist who also likes to build things.

## Key Projects

| Project | Description |
|---------|-------------|
| `bots/` | Healthcare bots — Dhanwantri (receptionist), Kavya (care pathway), Rohan (re-engagement) |
| `hermes-agent/` | Nous Research's self-improving AI agent (local git clone) |
| `hyperframes/` | TypeScript monorepo for video/frame processing |
| `skills/` | 36 OpenClaw skill directories for specialized tasks |
| `knowledge-base/` | Scraped content — X neurology scrapes, medical notes |
| `memory/` | Daily logs (110 dirs), long-term memory files |
| `content/` | Daily briefings, neuro education content, GMB posters |
| `pubmed-traditional-medicine-reviews/` | PubMed research workspace |

## Quick Reference

- **Telegram bot**: OpenClaw handles routing
- **X scraper**: `python3 x_scrape.py` (needs `~/.x_cookies.json`)
- **PubMed**: `cd ~/.openclaw/skills/pubmed-research && uv run python pubmed.py search "query"`
- **Hermes**: `cd heremes-agent && hermes`

## Active Configs

- `AGENTS.md` — workspace instructions
- `SOUL.md` — assistant personality
- `USER.md` — Bob's profile
- `MEMORY.md` — long-term memory
