## 📚 Today's Session (15 min — Thursday)

### Vocabulary (10 words to learn today)

| French | English | Memory trick |
|--------|---------|-------------|
| le sommeil | sleep | "somn-" like "insomnia" |
| le cerveau | brain | "cerveau" sounds like "serve" — your brain serves you |
| apprendre | to learn | "apprentice" — learning is apprenticeship |
| la mémoire | memory | "memory" — almost the same |
| le livre | book | "libro" — Spanish/Italian cognate |
| manger | to eat | "mange" in English (mange your food) |
| parler | to speak | "parley" — talking |
| comprendre | to understand | "comprehensive" — understanding |
| aujourd'hui | today | "au jour d'hui" = "on the day of today" |
| demain | tomorrow | "de main" — from hand (future) |

### Grammar Rule: French Sentence Structure

Basic order: **Subject → Verb → Object**

> Je mange une pomme. = I eat an apple.
> Tu bois de l'eau. = You drink water.

Question form: **Verb Swap** (flip subject + verb)

> Tu parles. → Parles-tu? (formal)
> Tu parles. → Est-ce que tu parles? (easier, spoken)

### Practice Sentence
Translate this:
> "I learn French today."
> ✅ **J'apprends le français aujourd'hui.**

Break it down:
- J' = I
- apprends = learn (from "apprendre")
- le français = French (the language)
- aujourd'hui = today
