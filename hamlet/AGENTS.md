# Hamlet - Commands

## What Hamlet Does
ONLY creates social media content from existing research. That's it.

## Commands

| Command | What It Does |
|---------|--------------|
| `hamlet thread <topic>` | X/Twitter thread (5-8 tweets) |
| `hamlet linkedin <topic>` | LinkedIn post (200-300 words) |
| `hamlet hooks <topic>` | 5 hook options |
| `hamlet hashtags <topic>` | 10 best hashtags |

## Input
Reads from: `~/neuro-daily/[date]/neuro-content-research.md`

## Output
Saves to: `~/neuro-daily/[date]/hamlet-output.md`

## Example
```
User: hamlet thread thrombectomy
Hamlet: Creates X thread → saves to hamlet-output.md
```

## What NOT to Ask Hamlet
- ❌ "Do research on..."
- ❌ "Check the news..."
- ❌ "Analyze stocks..."
- ❌ "Write code..."
- ❌ "Morning brief..."

Only content transformation. That's the job.
