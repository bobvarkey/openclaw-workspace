# Content Factory - Daily Workflow

## Overview
Automated content pipeline that runs every morning at 8 AM:
1. **Research Agent** → Finds top trending stories in your niche
2. **Writing Agent** → Writes scripts/threads from best ideas  
3. **Thumbnail Agent** → Generates AI images for content

## Channels (to be created in Discord)
- #research - Research agent posts trending opportunities
- #scripts - Writing agent posts drafts
- #thumbnails - Thumbnail agent posts AI images
