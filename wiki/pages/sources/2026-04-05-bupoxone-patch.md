# BUPOXONE — Buprenorphine Transdermal Patch

## Product Details

| Parameter | Value |
|-----------|-------|
| **Brand** | BUPOXONE (NRx — Predictably Potent) |
| **Active** | Buprenorphine |
| **Formulation** | Transdermal Patch |
| **Doses** | 5 mcg/hr, 10 mcg/hr |
| **Manufacturer** | ASTERA (Intas Pharmaceuticals) |
| **Reference** | S. Londhe et al., J Arthroplasty 35 (2020) S178-S181 |

## Indication

Treatment of **severe opioid responsive pain conditions** not adequately responding to non-opioid analgesics.

## Clinical Claim

"Safe, effective and predictable analgesia in POST OPERATIVE PAIN."
"Provides superior analgesia with lesser side effects vs conventional IV analgesics."

## Dosage & Administration

| Parameter | Value |
|-----------|-------|
| **Initial dose** | 5 mcg/hour (lowest dose) |
| **Patch change** | Every 7th day |
| **Max dose** | Two patches at same time |
| **Skin rotation** | Wait 3 weeks before same site |
| **Note** | Not suitable for acute pain |

## Contraindications

- Hypersensitivity to buprenorphine
- Delirium tremens

## Key Warnings

| Warning | Description |
|--------|-------------|
| **Hyperalgesia** | Opioid-induced hyperalgesia and allodynia possible |
| **Respiratory depression** | Risk in COPD, elderly, cachectic patients |
| **Adrenal insufficiency** | If suspected, treat with corticosteroids, wean opioid |
| **Severe hypotension** | Monitor during titration |
| **Intracranial pressure** | Caution in head injury, brain tumors |

## Side Effects (≥5%)

- Nausea, headache
- Application site pruritus
- Dizziness, constipation
- Somnolence, vomiting
- Application site erythema/rash
- Dry mouth

## Drug Interactions

| Interacting Drug | Risk |
|-----------------|------|
| Benzodiazepines | Respiratory depression |
| CYP3A4 inhibitors/inducers | Altered plasma concentrations |
| Serotonergic drugs | Serotonin syndrome |
| Mixed agonist/antagonists | Precipitated withdrawal |

## Special Populations

- **Pregnancy/Lactation**: Not recommended
- **Severe hepatic impairment**: Consider alternate analgesics
