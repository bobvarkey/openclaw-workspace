## Clarius L15 Scanner

### Overview
- **Wireless handheld ultrasound probe**
- **L15** = Linear 15 MHz probe (high frequency for superficial structures)
- Designed for **musculoskeletal (MSK)** applications
- Connects to mobile devices via WiFi

### Key Features

| Feature | Description |
|---------|-------------|
| **Wireless** | No cables — easy handling |
| **High frequency** | L15 = up to 15 MHz — ideal for MSK, nerves, vessels |
| **Preset modes** | Knee, MSK, vascular, etc. |
| **Mobile connectivity** | Streams to smartphone/tablet |
| **Battery powered** | Portable, bedside use |

### Clinical Applications

| Use | Application |
|-----|------------|
| **Musculoskeletal** | Tendons, ligaments, muscles |
| **Nerve imaging** | Peripheral nerves |
| **Vascular** | DVT, arteries, veins |
| **Point-of-care** | Bedside, emergency, MSK procedures |

### Controls
- **X**: Close
- **SC**: Scan/Freeze
- **Snowflake**: Freeze/Unfreeze
- **Gallery**: Saved images
- **Camera**: Capture
