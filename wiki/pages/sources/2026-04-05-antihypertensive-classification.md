# Antihypertensive Classification — ABCDE Framework

## ABCDE Framework

| Letter | Drug Class | Examples |
|--------|-----------|----------|
| **A** | ACE Inhibitors | Enalapril, Lisinopril, Ramipril |
| **A** | ARBs (Angiotensin Receptor Blockers) | Losartan, Valsartan, Telmisartan |
| **B** | Beta-blockers | Metoprolol, Atenolol, Carvedilol |
| **C** | Calcium Channel Blockers (CCB) | Amlodipine, Diltiazem, Nifedipine |
| **D** | Diuretics | HCTZ, Furosemide, Spironolactone, Eplerenone |
| **E** | (extra) Other/Secondary agents | |

## Class A — ACE Inhibitors & ARBs

| Subclass | Mechanism | Examples |
|---------|-----------|----------|
| **ACE-I** | Blocks angiotensin-converting enzyme | Enalapril, Lisinopril, Ramipril |
| **ARB** | Blocks AT1 receptor | Losartan, Valsartan, Telmisartan |
| **PC** (Primary Care) | First-line for uncomplicated HTN | ACE-I or ARB preferred |

## Class B — Beta-Blockers

| Notes | Indication |
|-------|-----------|
| Not first-line for uncomplicated HTN | Heart failure, post-MI, arrhythmia, migraine |
- Metoprolol, Atenolol, Carvedilol, Bisoprolol

## Class C — Calcium Channel Blockers

| Type | Examples |
|------|---------|
| **Dihydropyridine** (DHP) | Amlodipine, Nifedipine, Felodipine |
| **Non-DHP** | Diltiazem, Verapamil |

## Class D — Diuretics

| Type | Examples |
|------|---------|
| **Thiazide** | HCTZ, Chlorthalidone |
| **Loop** | Furosemide, Bumetanide |
| **K-sparing** | Spironolactone, Eplerenone, Amiloride |

## Peripheral vs Central Alpha Blockers

| Type | Examples | Notes |
|------|---------|-------|
| **Peripheral alpha blocker** | Prazosin, Doxazosin | Also used for BPH |
| **Central alpha-2 agonist** | Clonidine, Methyldopa | Sedation, rebound |

## Clinical Notes

- **First-line for most**: ACE-I/ARB + CCB or Diuretic
- **Black patients**: CCB or thiazide preferred over ACE-I
- **HF/CAD/MI**: ACE-I/ARB + Beta-blocker + Aldosterone antagonist
- **Diabetes**: ACE-I/ARB (renal protection)
