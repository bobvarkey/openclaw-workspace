# 4D DSA for AVMs — Temporal Imaging

## 4D DSA Concept

Combines temporal information with 3D geometry for dynamic AVM assessment.

## Components

| Component | Description |
|-----------|-------------|
| **Projection Images** | Multiple angle views |
| **Temporal Information** | Time-resolved contrast flow |
| **3D Reconstruction** | Geometry + time |
| **3D-DSA** | Standard 3D digital subtraction |
| **4D-DSA** | 3D + temporal resolution |

## 4D Imaging Protocol

| Parameter | Value |
|-----------|-------|
| **Flow rate** | 3 ml/s |
| **Volume** | 18 ml |
| **Rise time** | 0.1 s |
| **Xray delay** | 0 s |

## Clinical Applications

- Temporal resolution of arteriovenous shunting
- Flow velocity assessment
- Venous filling dynamics
- Treatment planning and follow-up
