# AVM Embolization — Pressure Cooker Technique

## Concept

"Pressure cook" — either antegrade or retrograde — to completely exclude the pathology.

## Pressure Cooker Technique Variants

### Case 1: Antegrade Pressure Cooker
For huge AVMs — approach from arterial feeders

### Case 2: Reverse Pressure Cooker (Transvenous Approach)
For selected cases — venous access to achieve embolization

## Embolic Materials Used

| Material | Role |
|---------|------|
| **Coil** | Vessel occlusion |
| **Glue (NBCO)** | Liquid embolic for penetration |
| **Onyx** | Controlled injection, penetration |

## Procedure Steps (Illustrated)

| Step | Description |
|------|-------------|
| Pre-embolization | 2D baseline angiogram |
| Catheter navigation | Detachable catheter with coil |
| Venous access | Foot of vein (red arrow) |
| Coiling | Coil deployment |
| Glue injection | Combined coil + glue |
| Cast formation | Embolic cast |
| Post-embolization | 2D post to confirm occlusion |
