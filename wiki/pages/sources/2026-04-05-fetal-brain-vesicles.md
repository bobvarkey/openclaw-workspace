# Fetal Brain Vesicles — Embryology

## Primary Brain Vesicles → Secondary Brain Vesicles

| Primary Vesicle | Secondary Vesicle |
|----------------|------------------|
| **Prosencephalon (forebrain)** | Telencephalon + Diencephalon |
| **Mesencephalon (midbrain)** | Mesencephalon (unchanged) |
| **Rhombencephalon (hindbrain)** | Metencephalon + Myelencephalon |

## Brain Vesicle Derivatives

| Secondary Vesicle | Adult Derivatives |
|------------------|------------------|
| **Telencephalon** | Cerebral hemispheres, basal ganglia, hippocampus |
| **Diencephalon** | Thalamus, hypothalamus, epithalamus, retina |
| **Mesencephalon** | Midbrain (superior/inferior colliculi, cerebral peduncles) |
| **Metencephalon** | Pons, cerebellum |
| **Myelencephalon** | Medulla oblongata |
| **Spinal cord** | Spinal cord |

## Choroidal Phase

- Stage: **11 weeks / 50 mm**
- Median Prosencephalic Vein (MPV) prominent
- Choroid plexus development begins
- Vascular development closely linked to ventricular system formation
