# Diaphragm Ultrasound — Probe Placement Technique

## Image Details

| Parameter | Value |
|-----------|-------|
| **Mode** | B-mode ultrasound |
| **Region** | Anterior axillary line, lower costal region |
| **Target** | Zone of Apposition (ZOA) |
| **Transducer** | Linear probe |

## Probe Placement Technique

### Location
- **Anterior axillary line**
- **Lower costal region**
- Over the Zone of Apposition (ZOA)

### Why This Location?
The **Zone of Apposition (ZOA)** is where:
- Diaphragm contacts the inner chest wall
- Contractile force is directed to expand rib cage
- Best area for thickness and function measurement

### Technique
1. Patient supine or semi-recumbent
2. Linear probe (high frequency, e.g., 10–15 MHz)
3. Place at **anterior axillary line**
4. Orient **caudocephalad** (angled upward toward diaphragm dome)
5. Identify the 3-layer diaphragm structure:
   - Hyperechoic: Pleura
   - Hypoechoic: Muscular portion
   - Hyperechoic: Peritoneum

## Key Landmarks

| Structure | Description |
|-----------|-------------|
| **Skin** | Superficial |
| **Intercostal muscles** | Between ribs |
| **ZOA** | Where diaphragm contacts chest wall |
| **Pleura** | Superior hyperechoic border |
| **Muscular diaphragm** | Hypoechoic center |
| **Peritoneum** | Inferior hyperechoic border |

## Clinical Application

This technique is used for:
- Measuring diaphragm **thickness**
- Calculating **thickening fraction**
- Assessing **paralysis vs normal function**
- Monitoring **phrenic nerve** recovery
