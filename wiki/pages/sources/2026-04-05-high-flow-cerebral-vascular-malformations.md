# High Flow Cerebral Vascular Malformations — Classification

## Classification

| Type | Subcategory |
|------|------------|
| **Choroidal** | VGAM (VGAD) — Vein of Galen Aneurysmal Malformation/Dilatation |
| **Pial** | pAVF — Pial Arteriovenous Fistula; pAVM — Pial AVM; CPA location |
| **Dural** | DSM — Dural Sinus Malformation; Paediatric Dural AVF |

## Key Entities

### VGAM (Choroidal Type)
- Most common high-flow perinatal cerebral vascular malformation
- Choroidal feeders → median prosencephalic vein

### Pial AVF (pAVF)
- Direct arteriovenous fistula without nidus
- Can occur at CPA (cerebellopontine angle)

### Pial AVM
- True nidus present
- Variable flow

### Paediatric Dural AVF
- Dural-based arteriovenous fistula in children
