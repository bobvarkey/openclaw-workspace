# Muscle Ultrasound

## Definition

A non-invasive imaging modality for assessing muscle structure, using high-frequency sound waves to evaluate muscle echogenicity, thickness, and architecture.

## Key Parameters

| Parameter | Description |
|-----------|-------------|
| **Thickness** | Muscle depth at anatomical landmarks |
| **Echogenicity** | Brightness of muscle on ultrasound |
| **Frame rate** | ≥20 fps for dynamic fibrillation detection |
| **Attenuation patterns** | Diffuse, focal, or moth-eaten |

## Heckmatt Scale

Standardized grading for muscle echogenicity:

| Grade | Description | Interpretation |
|-------|-------------|----------------|
| 1 | Normal | Healthy muscle |
| 2 | Slight hyperechogenicity | Mild changes |
| 3 | Texture disruption, some attenuation | Moderate myopathy/neurogenic |
| 4 | Texture unrecognizable, strong attenuation | Severe changes |

## Key Findings

### Spontaneous Activity
- **Fibrillations**: Detectable at ≥20 fps frame rate (previously only on EMG)
- **Fasciculations**: Visible involuntary twitches
- **Contraction fascics (CFs)**: Appear only with voluntary contraction

### Attenuation Patterns
- **Diffuse**: Throughout muscle
- **Focal**: Localized area
- **Moth-eaten**: Scattered hyper-attenuation
- **Spasticity**: May alter attenuation

## Advantages

- Non-invasive, no radiation
- Real-time dynamic imaging
- Cost-effective, portable
- Good for children
- Muscle-specific assessment

## Clinical Applications

- Myopathy assessment (Heckmatt grading)
- Neurogenic conditions (fibrillations, fasciculations)
- Sarcopenia assessment
- Diaphragm function (ZOA, thickening fraction)
- Body composition
