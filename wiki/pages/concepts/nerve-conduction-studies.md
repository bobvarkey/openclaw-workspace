# Nerve Conduction Studies (NCS)

## Definition

A core component of electrodiagnostic (EDX) medicine measuring how fast electrical signals travel through nerves. Part of the standard EMG/NCS examination alongside needle electromyography.

## Common Nerves Studied

- **Motor nerves**: median, ulnar, peroneal, tibial
- **Sensory nerves**: median, ulnar, sural
- **Cranial nerves**: hypoglossal, phrenic, facial

## Key Parameters

| Parameter | What It Measures |
|-----------|-----------------|
| Latency | Time from stimulation to response (ms) |
| Amplitude | Response size (mV for motor, µV for sensory) |
| Conduction velocity | Speed of signal propagation (m/s) |
| F-wave | Proximal nerve segment evaluation |
| H-reflex | Spinal reflex arc evaluation |

## Clinical Use

- Diagnosing carpal tunnel syndrome, ulnar neuropathy
- Evaluating diabetic peripheral neuropathy
- Assessing demyelinating diseases (CMT, CIDP)
- Monitoring neuromuscular junction disorders (myasthenia gravis, ALS)
