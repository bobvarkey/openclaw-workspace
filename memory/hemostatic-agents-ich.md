# Hemostatic Agents in ICH - Quick Reference

## Cryoprecipitate vs Fibrinogen Concentrate

| Feature | Cryoprecipitate | Fibrinogen Concentrate |
|---------|-----------------|----------------------|
| **Composition** | Multiple factors (Fibrinogen, FVIII, vWF, FXIII) | Purified fibrinogen only |
| **Source** | Human plasma | Purified/synthetic |
| **Dosing** | 1 unit raises fibrinogen ~70-100 mg/dL | Specific, consistent dose |
| **Thawing** | Required (30-60 min) | Ready to use |
| **ABO matching** | Required | Not needed |
| **Risk** | TRALI, allergic reactions | Lower reaction risk |
| **Cost** | Lower | Higher |

## Tranexamic Acid (TXA)

### Mechanism
**Antifibrinolytic** - blocks breakdown of blood clots.

- Binds to **plasminogen** 
- Prevents plasmin from breaking down fibrin
- Result: Clots stay intact longer

### In ICH Post-IVT
```
IVT → Alteplase → Plasminogen activated → Fibrin breakdown → Bleeding

TXA blocks this pathway → Bleeding stops
```

### Key Points
- **Timing**: Best given **early** (within 3-4.5 hours)
- **Dose**: 1g IV loading → 1g over 8 hours
- **Side effects**: DVT/PE risk (balanced in trauma/OBGYN)
- **Evidence**: CRASH-3 trial in TBI, various ICH studies

### When Used in ICH
1. Post-thrombolysis (IVT)
2. Primary ICH with ongoing bleeding
3. Surgical hemostasis adjunct

## Target Lab Values
- Fibrinogen >150 mg/dL
