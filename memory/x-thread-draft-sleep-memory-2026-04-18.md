So what determines good vs. bad microstructure?

Factors that DISRUPT sleep microstructure:
- Alcohol (severely fragments sleep patterns)
- Irregular sleep schedules
- Blue light before bed
- Stress/cortisol

Factors that ENHANCE it:
- Consistent sleep schedule
- Cool room (supports slow-wave sleep)
- Brief learning before bed (primes memory replay)
