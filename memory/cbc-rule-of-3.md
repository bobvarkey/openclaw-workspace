# CBC Rule of 3 - The Quiet Lie Detector

## The Rule

If the numbers agree, the CBC is likely accurate. If they don't, something is wrong.

### Primary Relationships
- **Hb × 3 ≈ Hct**
- **RBC × 3 ≈ Hb**

## What This Means

### When the Rule Holds
The numbers make physiological sense → CBC is reliable

### When the Rule Breaks

| Pattern | What It Suggests |
|---------|------------------|
| **Hb × 3 ≠ Hct** | Lab error, cold agglutinins, severe microcytosis |
| **RBC × 3 ≠ Hb** | Indices distortion, thalassemia trait, analytical artifact |

## Clinical Application

Most people read the numbers.

**Good clinicians check if the numbers agree with each other first.**

## Key Insight

The CBC Rule of 3 is a quick sanity check:
1. Multiply Hb by 3 → should ≈ Hct
2. Multiply RBC by 3 → should ≈ Hb

If these relationships don't hold, investigate before making clinical decisions.
