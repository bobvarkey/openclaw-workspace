| Key | Where to find |

| `X_BEARER_TOKEN` | Bearer Token |

### Step 2: Fill in credentials

```bash
nano ~/.openclaw/workspace/xmcp/.env
```

Paste in your values:
```
X_OAUTH_CONSUMER_KEY=your_key_here
X_OAUTH_CONSUMER_SECRET=your_secret_here
X_BEARER_TOKEN=your_bearer_token_here
X_OAUTH_CALLBACK_HOST=127.0.0.1
X_OAUTH_CALLBACK_PORT=8976
```

### Step 3: Start the MCP server

```bash
cd ~/.openclaw/workspace/xmcp
```
