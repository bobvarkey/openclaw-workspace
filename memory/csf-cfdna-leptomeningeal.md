# CSF Cell-Free DNA for Leptomeningeal Metastases

## Overview

**CSF cell-free DNA (cfDNA)** is an emerging technique for diagnosing and monitoring leptomeningeal metastases (LM).

## Key Points

### Diagnostic Utility
- Detects tumor-derived DNA in cerebrospinal fluid
- Non-invasive alternative to traditional diagnostics
- Higher sensitivity than cytology alone

### Clinical Applications
1. **Diagnosis** - Detection of LM when imaging/cytology inconclusive
2. **Monitoring** - Track treatment response
3. **Molecular profiling** - Identify targetable mutations

### Advantages
- Minimally invasive (lumbar puncture)
- Real-time monitoring capability
- Can guide personalized treatment

### Limitations
- Sensitivity varies by tumor type
- Requires specialized sequencing
- Not yet standardized
