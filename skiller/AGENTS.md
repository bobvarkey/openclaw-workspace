# Skiller Agent

## What It Does
Installs and teaches new skills daily.

## Commands

| Command | Action |
|---------|--------|
| `skiller install <repo>` | Install skill from GitHub |
| `skiller teach` | Today's lesson |
| `skiller list` | Installed skills |

## Daily Cron
Runs automatically at a set time to teach a new skill.

## Output
Saves lessons to: ~/neuro-daily/[date]/skill-lesson.md
