# Comprehensive Clinical Examination Protocol
**Outside → Inside Systematic Framework**
*Created: 2026-03-18*

---

## 🧠 LAYER 1: MIND (Mental Status Examination)

### 1.1 General Appearance & Behavior
- [ ] Appearance: grooming, clothing, posture
- [ ] Psychomotor activity: agitated, retarded, normal
- [ ] Eye contact: present, absent, inconsistent
- [ ] Rapport: cooperative, guarded, hostile

### 1.2 Consciousness & Orientation
- [ ] Level of consciousness: alert, drowsy, stuporous, comatose
- [ ] GCS (E___V___M___ = ___/15)
- [ ] Orientation: time / place / person
- [ ] Attention: digit span (forward/backward), serial 7s

### 1.3 Cognition (Screening)
- [ ] MMSE (/30) or MoCA (/30)
- [ ] Memory: immediate recall, short-term, long-term
- [ ] Executive function: Luria sequences, set-shifting
- [ ] Visuospatial: clock drawing, cube copy

### 1.4 Mood & Affect
- [ ] Mood: patient's reported state
- [ ] Affect: observed (congruent/incongruent)
- [ ] Range: full, restricted, blunted, flat

### 1.5 Thought Process & Content
- [ ] Stream: pressured, poverty of speech, flight of ideas
- [ ] Form: logical, tangential, circumstantial, loose associations
- [ ] Content: delusions, suicidal/homicidal ideation, obsessions

### 1.6 Perception
- [ ] Hallucinations: auditory, visual, tactile, olfactory
- [ ] Illusions, depersonalization, derealization

### 🚨 RED FLAGS
- Acute confusion/delirium → urgent workup
- Active suicidal ideation → safety protocol
- GCS ≤8 → airway protection

### 2.4 Sensory System
- **Light touch:** dermatomes (C5-T1, L3-S1)
- **Pinprick:** spinothalamic tract
- **Vibration:** 128Hz tuning fork (digits, malleoli)
- **Proprioception:** joint position sense (digits)
- **Cortical:** two-point discrimination, stereognosis, graphesthesia

### 2.5 Cerebellar Function
- [ ] Finger-nose-finger (intention tremor, past-pointing)
- [ ] Heel-shin (ataxia)
- [ ] Rapid alternating movements (dysdiadochokinesia)
- [ ] Rebound test
- [ ] Gait: tandem walk, Romberg test

### 2.6 Gait Assessment
- [ ] Normal walking
- [ ] Heel walking (L4-L5)
- [ ] Toe walking (S1-S2)
- [ ] Tandem gait (midline cerebellar)
- [ ] Romberg (proprioception vs cerebellar)

### 🚨 RED FLAGS
- Acute focal deficit → stroke protocol
- Bilateral weakness + sensory level → cord compression
- Papilledema + headache → raised ICP
- Bilateral upgoing plantars → upper motor neuron lesion

## 🧴 LAYER 3: SKIN & INTEGUMENTARY

### 3.1 Inspection (Systematic)
- [ ] Color: pallor, cyanosis, erythema, jaundice
- [ ] Lesions: macule, papule, nodule, vesicle, ulcer, plaque
- [ ] Distribution: dermatomal, flexural, extensor, symmetrical
- [ ] Scars: surgical, trauma, self-harm

### 3.2 Palpation
- [ ] Temperature: warm, cool (asymmetry?)
- [ ] Moisture: dry, diaphoretic
- [ ] Turgor: normal, decreased (dehydration)
- [ ] Texture: smooth, rough, indurated

### 3.3 Nails
- [ ] Clubbing
- [ ] Koilonychia (iron deficiency)
- [ ] Splinter hemorrhages
- [ ] Leukonychia
- [ ] Capillary refill time (<2 seconds)

### 3.4 Hair
- [ ] Distribution: hirsutism, alopecia
- [ ] Texture: brittle (hypothyroid), fine (hyperthyroid)

### 3.5 Dermatome Mapping
- [ ] Check for sensory level (if neurological symptoms)
- [ ] Herpes zoster distribution

### 🚨 RED FLAGS
- Petechial rash + fever → meningococcemia
- Necrotizing fasciitis → surgical emergency
- Erythema migrans → Lyme disease

## 👁️ LAYER 4: HEENT (Head, Eyes, Ears, Nose, Throat)

### 4.1 Head
- [ ] Skull shape, symmetry
- [ ] Scalp: tenderness, masses, temporal artery thickening
- [ ] TMJ: clicking, tenderness, ROM

### 4.2 Eyes
- [ ] Visual acuity (Snellen chart)
- [ ] Visual fields: confrontation test
- [ ] Pupils: size, shape, symmetry, RAPD (swinging torch)
- [ ] Eye movements: cover/uncover test, nystagmus
- [ ] Fundoscopy: papilledema, disc pallor, hemorrhages, exudates
- [ ] Intraocular pressure (if indicated)

### 4.3 Ears
- [ ] Otoscopy: canal, TM integrity
- [ ] Hearing: whisper test, finger rub
- [ ] Weber (512Hz): lateralization
- [ ] Rinne: AC > BC (normal)

### 4.4 Nose & Sinuses
- [ ] External symmetry
- [ ] Nasal mucosa: color, discharge, polyps
- [ ] Sinus tenderness: frontal, maxillary

### 4.5 Throat & Oral Cavity
- [ ] Lips: color, lesions
- [ ] Tongue: movement, coating, lesions
- [ ] Pharynx: tonsils, erythema, exudate
- [ ] Neck: lymph nodes, thyroid, JVP, tracheal position

### 🚨 RED FLAGS
- Temporal artery tenderness + age >50 → giant cell arteritis
- Painless vision loss → CRAO
- Bilateral papilledema → raised ICP

## 🩸 LAYER 5: VASCULAR SYSTEM

### 5.1 Peripheral Vascular
- [ ] Inspection: skin color, ulcers, varicose veins, hair loss
- [ ] Temperature: asymmetry
- [ ] Capillary refill: <2 seconds

### 5.2 Pulse Assessment (Rate, Rhythm, Volume, Character)

| Brachial | | | |
| Femoral | | | |
| Popliteal | | | |
| Dorsalis pedis | | | |
| Posterior tibial | | | |

### 5.3 Bruits
- [ ] Carotid (systolic)
- [ ] Abdominal aorta
- [ ] Renal arteries
- [ ] Femoral

### 5.4 Edema Assessment
- [ ] Location: dependent, sacral, periorbital
- [ ] Pitting vs non-pitting
- [ ] Grade (1+ to 4+)
- [ ] Unilateral vs bilateral

### 5.5 Special Tests
- [ ] Ankle-Brachial Index (ABI): normal 1.0-1.4
- [ ] Buerger's test (leg elevation → pallor)
- [ ] Allen's test (collateral hand circulation)

### 🚨 RED FLAGS
- Acute limb ischemia (6 P's: pain, pallor, pulselessness, paresthesia, paralysis, poikilothermia)
- Unilateral leg swelling → DVT (Wells score)
- Expansile pulsatile mass → AAA

## ❤️ LAYER 6: CARDIOVASCULAR

### 6.1 Inspection
- [ ] Chest wall deformities
- [ ] Visible apex beat
- [ ] Jugular venous pressure (JVP)
- [ ] Scars (previous surgery)

### 6.2 Palpation
- [ ] Apex beat: location, character
- [ ] Thrills (palpable murmurs)
- [ ] Heaves (right ventricular hypertrophy)

### 6.3 Auscultation (4 areas: Aortic, Pulmonic, Tricuspid, Mitral)

| Area | S1 | S2 | Extra sounds | Murmur |
|------|----|----|-------------|--------|
| Aortic (2nd RICS) | | | | |
| Pulmonic (2nd LICS) | | | | |
| Tricuspid (LLEB) | | | | |
| Mitral (5th MICS) | | | | |

- [ ] Murmur characterization: timing, location, radiation, grade (I-VI), quality
- [ ] Dynamic maneuvers: Valsalva, squatting, inspiration

### 6.4 Peripheral Signs
- [ ] Pulse deficit
- [ ] Blood pressure (both arms if indicated)
- [ ] Capillary refill
- [ ] Peripheral edema
- [ ] Splinter hemorrhages, Janeway lesions, Osler nodes

### 🚨 RED FLAGS
- New systolic murmur + chest pain → urgent echo
- Diastolic murmur → always pathological
- Chest pain + hemodynamic instability → ACS protocol

## 🫁 LAYER 7: RESPIRATORY

### 7.1 Inspection
- [ ] Respiratory rate and pattern
- [ ] Use of accessory muscles
- [ ] Chest shape: barrel chest, pectus deformities
- [ ] Symmetry of chest expansion
- [ ] Cyanosis: central vs peripheral

### 7.2 Palpation
- [ ] Chest expansion (symmetry)
- [ ] Tactile vocal fremitus
- [ ] Tracheal position

### 7.3 Percussion
- [ ] Resonant (normal)
- [ ] Dull (consolidation, effusion)
- [ ] Hyper-resonant (pneumothorax, emphysema)

### 7.4 Auscultation

| Area | Right | Left | Notes |
|------|-------|------|-------|
| Upper zones | | | |
| Middle zones | | | |
| Lower zones | | | |

- [ ] Breath sounds: vesicular, bronchial breathing
- [ ] Added sounds: crackles (fine/coarse), wheeze, rhonchi, pleural rub
- [ ] Vocal resonance: increased (consolidation), decreased (effusion)

### 🚨 RED FLAGS
- Stridor → upper airway obstruction
- Unilateral absent breath sounds → pneumothorax/effusion
- Silent chest in asthma → life-threatening

## 🍽️ LAYER 8: ABDOMINAL & GI

### 8.1 Inspection
- [ ] Shape: scaphoid, distended, asymmetric
- [ ] Scars, stomas, dilated veins
- [ ] Visible peristalsis, hernias
- [ ] Umbilicus: position, herniation (Sister Mary Joseph node)

### 8.2 Auscultation (BEFORE palpation)
- [ ] Bowel sounds: normal, increased, absent
- [ ] Bruits: aortic, renal, iliac
- [ ] Rubs: liver, spleen

### 8.3 Percussion
- [ ] Liver span (MCL): ___cm
- [ ] Splenic percussion (Traube's space)
- [ ] Shifting dullness (ascites)
- [ ] Bladder distension

### 8.4 Palpation
- **Light palpation:** tenderness, guarding, rigidity
- **Deep palpation:** masses, rebound tenderness
- **Organ assessment:**

| Organ | Palpable | Size | Tenderness | Notes |
|-------|----------|------|------------|-------|
| Liver | | | | |
| Spleen | | | | |
| Kidneys | | | | |
| Aorta | | | | |

### 8.5 Special Signs
- [ ] Murphy's sign (cholecystitis)
- [ ] McBurney's point (appendicitis)
- [ ] Rovsing's sign (appendicitis)
- [ ] Psoas sign / Obturator sign (retrocecal appendix)
- [ ] CVA tenderness (renal pathology)

### 🚨 RED FLAGS
- Rigid abdomen → peritonitis → surgical emergency
- Pain out of proportion to exam → mesenteric ischemia
- Pulsatile expansile mass >3cm → AAA rupture risk

## 🚻 LAYER 9: GENITOURINARY & REPRODUCTIVE

### 9.1 Male
- [ ] External genitalia: inspection
- [ ] Hernial orifices
- [ ] DRE: prostate size, consistency, nodules
- [ ] Testes: masses, tenderness

### 9.2 Female
- [ ] External genitalia: inspection
- [ ] Speculum: cervix, vaginal walls
- [ ] Bimanual: uterus, adnexa, fornices
- [ ] Cervical screening if due

### 9.3 Urinary Assessment
- [ ] History: frequency, urgency, nocturia, hesitancy, terminal dribbling
- [ ] Incontinence type: stress, urge, overflow
- [ ] Post-void residual (if indicated)
- [ ] Urinalysis: dipstick for blood, protein, glucose, leukocytes

### 🚨 RED FLAGS
- Acute urinary retention → catheterize
- Testicular torsion → surgical emergency (pain <6 hours)
- Post-menopausal bleeding → malignancy until proven otherwise

## 🦴 LAYER 10: MUSCULOSKELETAL

### 10.1 Regional Joint Examination (for each affected joint: LOOK → FEEL → MOVE → SPECIAL TESTS)

#### Shoulder
- [ ] Inspection: asymmetry, swelling, wasting
- [ ] Palpation: AC joint, bicipital groove, supraspinatus
- [ ] ROM: flexion (180°), abduction (180°), internal/external rotation
- [ ] Special: Neer's, Hawkins, empty can, apprehension

#### Elbow
- [ ] Inspection: olecranon bursitis, rheumatoid nodules
- [ ] Palpation: medial/lateral epicondyles, olecranon
- [ ] ROM: flexion (150°), extension (0°), pronation/supination
- [ ] Special: tennis elbow, golfer's elbow tests

#### Wrist & Hand
- [ ] Inspection: swelling, deformity (swan neck, boutonnière)
- [ ] Palpation: joints, thenar/hypothenar eminences
- [ ] ROM: grip strength, pinch, finger extension/flexion
- [ ] Special: Phalen's, Tinel's, Finkelstein's

#### Spine
- [ ] Inspection: scoliosis, kyphosis, lordosis
- [ ] Palpation: spinous processes, paraspinal muscles
- [ ] ROM: flexion (80°), extension (25°), lateral flexion (35°), rotation (30°)
- [ ] Special: straight leg raise (L4-S1), femoral nerve stretch (L2-L4)

#### Hip
- [ ] Gait: Trendelenburg, antalgic
- [ ] ROM: flexion (120°), extension (20°), abduction/adduction, rotation
- [ ] Special: Thomas test, FABER/Patrick's, log roll

#### Knee
- [ ] Inspection: effusion, valgus/varus
- [ ] Palpation: joint line, patella, popliteal fossa
- [ ] ROM: flexion (135°), extension (0°)
- [ ] Special: anterior/posterior drawer, Lachman's, McMurray's, valgus/varus stress

#### Ankle & Foot
- [ ] Inspection: swelling, flat foot, claw toes
- [ ] Palpation: Achilles tendon, tarsal bones
- [ ] ROM: dorsiflexion (20°), plantarflexion (50°), inversion/eversion
- [ ] Special: Thompson test, anterior drawer

### 10.2 Spine Examination
- [ ] Schober's test (lumbar mobility)
- [ ] Chest expansion (<5cm = ankylosing spondylitis)
- [ ] Cauda equina: perianal sensation, anal tone, urinary retention

### 🚨 RED FLAGS
- Hot, red, swollen monoarthritis → septic arthritis → aspirate urgently
- Cauda equina syndrome → emergency MRI
- Open fracture → orthopedic emergency

## 🧪 LAYER 11: ENDOCRINE SCREENING

### 11.1 Thyroid
- [ ] Inspection: goiter, eye signs (exophthalmos, lid lag)
- [ ] Palpation: size, nodules, tenderness, lymph nodes
- [ ] Auscultation: bruit (thyrotoxicosis)
- [ ] Signs:
  - Hyperthyroidism: tremor, tachycardia, warm moist skin, brisk reflexes
  - Hypothyroidism: dry skin, bradycardia, delayed relaxation of reflexes, periorbital puffiness

### 11.2 Adrenal
- [ ] Cushing's: central obesity, moon face, striae, buffalo hump, easy bruising
- [ ] Addison's: hyperpigmentation, hypotension, vitiligo
- [ ] Postural BP drop (>20/10 mmHg)

### 11.3 Pituitary
- [ ] Visual fields (bitemporal hemianopia)
- [ ] Acromegaly: enlarged hands/feet, prognathism, frontal bossing
- [ ] Panhypopituitarism: loss of secondary sexual characteristics

### 11.4 Metabolic Screening
- [ ] Acanthosis nigricans (insulin resistance)
- [ ] Xanthelasma (hyperlipidemia)
- [ ] BMI, waist circumference

### 🚨 RED FLAGS
- Thyroid storm: fever + tachycardia + altered consciousness
- Addisonian crisis: hypotension + hyponatremia + hyperkalemia
- Pheochromocytoma: paroxysmal hypertension + headache + sweating + palpitations

## 📋 QUICK REFERENCE: EXAMINATION FLOW

### Standard Order (Outside → Inside)
1. **Mind** — Mental status, cognition, mood
2. **Brain & Nerves** — Cranial nerves, motor, sensory, cerebellar
3. **Skin** — Inspection, palpation, nails, hair
4. **HEENT** — Eyes, ears, throat, thyroid
5. **Vascular** — Pulses, bruits, edema
6. **Cardiovascular** — Heart sounds, murmurs
7. **Respiratory** — Lungs (IPPA)
8. **Abdominal** — GI (IAPA)
9. **Genitourinary** — Reproductive, urinary
10. **Musculoskeletal** — Joints, spine
11. **Endocrine** — Thyroid, adrenal, metabolic
12. **Lymphatic** — Nodes, spleen, immune

### Time Estimates
- Full screening exam: 15-20 minutes
- Focused system exam: 5-10 minutes
- Emergency neuro exam: 2-3 minutes
